# E-Commerce-App

## Main-Advantages: 
- Online-Payment-Gateway with Stripe

- Generating Invoice & send it via Email after payment

- User-Cart

- Wish-List

- Authentication:
  - Register
  - Login
  - Forget Password
  - Reset Password
  - Send OTP via Email

- Validation with Yup

- Form handling using Formik

- Protected Routing

- Axios to fetch data from the API.

## Extra-Packages:
  - react-detect-offline
  - react-hot-toast
  - slick-carousel
  - react-helmet.

## Technologies:
- ReactJS
- ContextAPI
- Bootstrap5
- HTML5
- CSS3
- JavaScript
- LocalStorage
